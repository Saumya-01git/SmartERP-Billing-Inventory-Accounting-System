# SmartERP

SmartERP is a Tally-inspired web application for billing, inventory, accounting, customers, suppliers, sales, purchases, and reports.

## MVP Features
- User Login/Register
- Company Management
- Customer Ledger
- Supplier Ledger
- Stock Items
- Purchase Voucher
- Sales Voucher
- Stock Auto Update
- PDF Invoice
- Basic Reports

## Tech Stack
Frontend: React + Vite + Tailwind CSS  
Backend: Node.js + Express  
Database: PostgreSQL  
Authentication: JWT
